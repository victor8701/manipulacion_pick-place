# Manipulación: Pick & Place

Este repositorio contiene las tareas y ejercicios relacionados con la asignatura de Manipulación Robótica, centrados en estrategias de Pick & Place.

## Estructura del Repositorio

- **Tarea1/**: Implementación inicial de trayectorias y agarres básicos.
- **Tarea2/**: Evolución de la Tarea 1 incorporando el control de un brazo robótico completo.

## Requisitos
- ROS (Noetic/Melodic)
- Gazebo
- Dependencias de `manipulacion_pkg`: https://github.com/elisabeth-ms/manipulacion_pkg
