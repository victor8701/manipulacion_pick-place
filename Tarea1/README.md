# Tarea 1: Entrenamiento de Agarres y Trayectorias

Esta carpeta contiene los scripts y notebooks necesarios para la primera fase del proyecto de manipulación.

## Contenido
- `manipulacion_tarea1.ipynb`: Notebook con el desarrollo y pruebas de la tarea.
- `manipulacion_tarea1_entrenamiento.py`: Script de entrenamiento para optimizar los puntos de agarre.

## Objetivos
1. Definir posiciones de agarre (grasp poses) para diferentes objetos.
2. Simular el cierre del gripper en Gazebo.
3. Evaluar la estabilidad del agarre.
